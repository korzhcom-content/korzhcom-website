GETTING STARTED WITH LOCALIZER

Thank you for installing Localizer. We hope you will find it useful and handy. To make your start easier, we have prepared several simple and short instructions:
  1. First of all you may want to see Localizer at work. To do it - try to open one of our sample applications to learn how it is organized. The source codes for Localizer's samples are available in the [My Documents]\KorzhLocalizer\Samples\ folder (you can open this folder by selecting "Open Samples folder" menu item in Localizer's Start menu). Just open one of the sample projects in your Delphi IDE and compile it. When it's compiled and run well - then try to switch the language and see if it works well.
  2. After that you can try to apply Localizer to your own project. Read the following knowledge base article to get an idea how to start with this task.
  3. To proceed further with your project localization you can read How to translate using Language Manager and How to use Localizer in your code articles.
Please note! if anything of described above went wrong - don't hesitate and contact us immediately. Our support team will be glad to help you with any question.
