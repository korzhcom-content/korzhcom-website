GETTING STARTED WITH
SIMPLE QUERY

Thank you for installing a Simple Query library. We hope you will find it useful and handy. To make your start easier, we have prepared several simple and short instructions:
  1. First of all you may want to see Simple Query at work. To do it - try to open one of our sample applications to learn how it is organized. The source codes for Simple Query's samples are available in the [InstallDir]\sq\demos\ folder Just open one of the sample projects in your Delphi IDE and compile it. The most handy demo project to start with is KQB. When it's compiled and run well - then try to change or to add some conditions and/or columns, load another query and see how it works.
  2. After that you can try to apply Simple Query to your own project. Just read "Getting started" topic in SimpleQuery's help file and follow the instructions placed there.
Please note! if anything of described above went wrong - don't hesitate and contact us immediately. Our support team will be glad to help you with any question.
